#### 在Git中，origin / master与origin master之间有什么区别？这里实际上有三件事：origin master是两件事，origin/master一件事。共计三件事。

两个分支：

* master 是一个本地分支

* origin/master是远程分支（它是名为“origin” 的远程分支的本地副本，名为“master”）
  一个遥控器：

* origin 是一个遥远的
